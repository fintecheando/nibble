PVA 2.0 Seguridad
2.2 Usuarios

 

 

 

I. DESCRIPCIÓN GRAL

 

 

 

Sirve para dar de alta a los usuarios que tendrán acceso al sistema de acuerdo al puesto y perfil que tengan en la refaccionaría

 

 

 

II. ELEMENTOS

 

 

 

 

1.- Selección

2.- Usuario

3.- Descripción

4.- Nombre usuario

5.- Identificador

6.- Grupo

7.- Contraseña

8.-Verificar contraseña

9.- Usuario habilitado

10.- Descripción

11.- Crear

12.- Modificar

13.- Eliminar

14.- Restablecer

 

 

II. FUNCIONALIDAD

 

a.)        Dar de alta a los usuarios deacuerdo a los puestos establecidos que tiene el sistema

b.)       Modificar el tipo de usuario

c.)        Eliminar usuario

 

PASOS PARA

 

 

 

 

I.- DAR DE ALTA UN USUARIO

 

Þ        Escribir nombre del usuario

Þ        Escribir identificador ( Es el nombre con el que se registran para entrar al sistema)

Þ        Seleccionar grupo de usuario ( Es el conjunto de categorías en las que se divide el sistema que son: Senior, Junior, Crédito, Vendedor y Caja)

Þ        Escribir contraseña ( Es la palabra y/o número clave secreta para tener acceso)

Þ        Verificar contraseña

Þ        Dar clic en usuario habilitado ( Al dar de alta un usuario es necesario que se habilité o active ya que solo así podrá tener acceso al sistema)

Þ        Escribir descripción de la función que realiza la persona dentro de la refaccionaría

Þ        Por ultimo dar clic en crear

 

 

2.- MODIFICAR UN USUARIO

 

Þ        Seleccionar el usuario

Þ        Cambiar los datos

Þ        Hacer clic en modificar

 

 

3.- ELIMINAR UN USUARIO

 

Þ        Seleccionar el usuario

Þ        Hacer clic en eliminar
